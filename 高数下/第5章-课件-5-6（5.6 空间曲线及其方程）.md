## 三、空间曲线的参数方程

$$
\left\{\begin{array}{l}
x=x(t) \\
y=y(t) \\
z=z(t)
\end{array} \quad\right. \text { 空间曲线的参数方程 }
$$

当给定 $\boldsymbol{t}=\boldsymbol{t}_{1}$ 时，就得到曲线上的一个点 $\left(x_{1}, y_{1}, z_{1}\right)$ ，随着参数的变化可得到曲线上的全部点。

例5．将下列曲线化为参数方程表示：
（1）$\left\{\begin{array}{l}x^{2}+y^{2}=1 \\ 2 x+3 z=6\end{array}\right.$

$$

$$
\left\{\begin{array}{l}

y=\frac{a}{2} \sin t \\
z=a \sqrt{\frac{1}{2}-\frac{1}{2} \cos t}
\end{array} \quad(0 \leq t \leq 2 \pi)\right.
$$

## 空间曲线小结与思考题

空间曲线的一般方程、参数方程．

$$
\left\{\begin{array} { l } 
{ F ( x , y , z ) = 0 } \\
{ G ( x , y , z ) = 0 }
\end{array} \quad \left\{\begin{array}{l}
x=x(t) \\
y=y(t) \\
z=z(t)
\end{array}\right.\right.
$$

空间曲线在坐标面上的投影．

$$
\left\{\begin{array} { l } 
{ H ( x , y ) = 0 } \\
{ z = 0 }
\end{array} \left\{\begin{array} { l } 
{ R ( y , z ) = 0 } \\
{ x = 0 }
\end{array} \left\{\begin{array}{l}
T(x, z)=0 \\
y=0
\end{array}\right.\right.\right.
$$
