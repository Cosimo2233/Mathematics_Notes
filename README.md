# 高等数学笔记

这是一个按课程章节整理的高等数学 Markdown 笔记仓库，内容覆盖极限与连续、一元微积分、无穷级数、空间解析几何、多元微积分和常微分方程等主题。适合按课程顺序阅读，也可以作为复习时的公式、例题和方法索引。

## 目录结构

```
.
├── 01-极限与连续              (8 篇)
├── 02-一元函数微分学          (13 篇)
├── 03-一元函数积分学          (15 篇)
├── 04-无穷级数                (8 篇)
├── 05-空间解析几何与向量代数  (5 个主题，已合并)
├── 06-多元函数微分学          (10 个主题，已合并)
├── 07-多元函数积分学          (15 个主题，已合并)
├── 08-常微分方程              (8 个主题，已合并)
└── README.md
```

### 各章内容

- **01-极限与连续**：函数、数列极限、函数极限、无穷小与连续性、习题课
- **02-一元函数微分学**：导数、求导法则、中值定理、泰勒公式、罗必塔法则和函数性态、习题课
- **03-一元函数积分学**：不定积分、换元积分法、定积分、微积分基本定理、几何应用和广义积分、习题课
- **04-无穷级数**：常数项级数、函数项级数、幂级数和 Fourier 级数、习题课
- **05-空间解析几何与向量代数**：
  - 向量与空间直角坐标系
  - 数量积、向量积与混合积
  - 平面与空间直线
  - 曲面及其方程
  - 空间曲线及其方程
- **06-多元函数微分学**：
  - 多元函数的概念、偏导数、全增量与全微分
  - 复合函数的微分法、隐函数及其微分法
  - 方向导数与梯度、切线法平面与切平面法线
  - 二元函数的 Taylor 公式、多元函数极值与 Lagrange 乘数法
  - 习题课
- **07-多元函数积分学**：
  - 重积分的概念与性质、二重积分、三重积分
  - 重积分的换元法、重积分的几何应用
  - 第一/二类曲线积分、第一/二类曲面积分
  - Green 公式、Gauss 公式、Stokes 公式
  - 曲面及方程补充、习题课
- **08-常微分方程**：
  - 微分方程概念与可分离变量、齐次方程与一阶线性方程
  - Bernoulli 方程与全微分方程、可降阶方程与解的结构
  - 二阶线性常系数方程、常系数非齐次方程与 Euler 方程
  - 微分方程的简单应用、习题课

## 阅读建议

建议从一级编号目录按顺序阅读。每个目录下的文件名以两位编号开头，表示该章内部的学习顺序；部分章节还包含"习题课""小结"等复习材料。

笔记中包含 LaTeX 公式。推荐使用支持数学公式渲染的 Markdown 编辑器或笔记工具阅读，例如 VS Code 搭配 Markdown Preview Enhanced，或其他支持 MathJax/KaTeX 的阅读环境。

## 维护约定

- 新增正式笔记时，优先放入对应的编号章节目录。
- 文件名尽量保持 `两位编号-主题.md` 的形式，便于排序和检索。
- 公式统一使用 Markdown 中的 LaTeX 写法，行间公式使用 `$$ ... $$`。
- 05~08 章的笔记按主题合并存放在子目录中，每个子目录对应一个合并文件，不再拆分过细。
- README 与面向读者的说明文档使用中文编写。
